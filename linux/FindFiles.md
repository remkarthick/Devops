## List Files in Linux, biggest File will be at the bottom

`
ls -lrSh
`

> l - long list
> r - reverse
> S - Size
> h - human readable

## List Files in Linux, sort it by reverse time

`
ls -lrt
`

> l - long list
> r - reverse
> t - time


## List Files in Linux, sort it by reverse time
